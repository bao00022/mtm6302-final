## Design Concept - Part 2
The design concept of this webpage centers on creating an image-focused immersive experience. Since astronomy images are typically presented in deep tones of blue and black, a light blue shade was chosen as the background. This not only maintains harmony within the same color family but also provides sufficient contrast to highlight the images themselves. In terms of layout, the main image occupies a larger portion of the page to ensure clear visual focus. The interaction flow is kept simple—users can select images by date and save their favorites to a dedicated “Favorite” section for easy access and management later. Overall, the design aims to deliver a clean, intuitive, and visually engaging user experience.
